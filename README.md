A Python client for accessing a [textrepo](https://github.com/knaw-huc/textrepo) server

# installing

### using poetry

```commandline
poetry add annorepo-client
```

### using pip

```commandline
pip install annorepo-client
```

### locally, after checking out this repo first

```commandline
pip install .
```
